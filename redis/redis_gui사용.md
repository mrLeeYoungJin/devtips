redis 클라이언트 오픈소스(medis)
==================

> medis site
- https://github.com/luin/medis

#### 설치 - 사전에 npm 설치 해야 함

> 소드 다운로드

```
$ git clone https://github.com/luin/medis.git
```

> install

```
$ npm install
```

> compile

```
$ npm run-script build
```

> start
##### 서비스가 실행 되면 electron 으로 어플리케이션 생성이 되어 클라이언트 프로그램이 자동 실행 됨

```
$ npm start
```
