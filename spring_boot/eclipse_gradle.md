spring boot + eclipse_gradle 실행
=================================

#### spring boot project download

```
command 실행
spring boot project 가 있는 폴더로 이동
$ gradlew build

* gradle test 파일이 없어서 fail 시 명령어 추가
$ gradlew build -x test

$ gradlew eclipse
```
